An e-plantShopping application developed using React.js.
The project is to create a shopping cart application for an online plant shop that offers a variety of house plants. It should include three pages: a landing page, a product listing page, and a shopping cart page.

The Landing Page -
    The landing page contains:
        * a background image
        * a paragraph about the company
        * the company name
        * a Get Started button linking tot he product page

The Product Listing Page -
    The product listing page contains houseplants organized into categories. Each plant has the following details:
        * a thumbnail image
        * plant name
        * price
        * an Add to Cart button

The Shopping Cart Page -
    The shopping cart page displays details about the items in it, including the total number of plants in the cart, the total cost of all items in the cart, a continue shopping button, and a checkout button.
